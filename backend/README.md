# Furniture Store Backend

## Setup
1. Ensure you have MongoDB installed and running locally.
2. Navigate to this directory: `cd backend`
3. Install dependencies: `npm install`
4. Seed the database: `node seed.js`
5. Start the server: `npm start` or `npm run dev`

## API Endpoints
- `GET /api/products`: Get all products.
- `POST /api/products/:id/like`: Like a product.
